Силиконовые приманки

Подбор оснастки: грузы, крючки, монтажи, рыба

Выбери условия рыбалки — получишь рекомендацию по оснастке. Все параметры опциональны кроме первых двух.

Полный справочник по всем размерам и типам приманок. На мобильном — карточный вид.

О проекте

Инструмент помогает подобрать оснастку для джиговой ловли на силикон: вес груза, тип и размер крючка, монтаж, проводку.

База данных собрана из открытых источников (производители крючков, обзоры практиков) и экспертных рекомендаций. Покрывает диапазон от микроджига (0.5 г) до классики (до 40 г).

Контекст применимости

  • Регион: Россия, средняя полоса. Реки и водохранилища.
  • Целевые виды: окунь, судак, щука, голавль, язь, берш, жерех.
  • Техника: от микроджига до классического джига.
  • Типичная глубина: 1.5–4 м (рекомендации корректируются под другие условия).
  • Течение: от стоячей воды до умеренного (с поправкой для сильного).

Как работает конфигуратор

  1. По размеру и типу приманки находится базовая запись в БД
  2. Базовый вес груза умножается на коэффициенты глубины и течения
  3. Результат округляется до ближайшего стандартного веса чебурашки
  4. По условиям дна определяется приоритетный крючок (одинарник vs офсет)
  5. Выбирается оптимальный тип монтажа для комбинации условий
  6. Сверяется выбранная рыба со списком целевой для этой приманки
  7. Проверяются «нерабочие» комбинации (например, 1″ виброхвост на щуку)
  8. Выводится рекомендация с обоснованием каждого пункта

Рекомендуемые крючки

Принципы выбора крючка

  • На приманках до 1″ — только открытый одинарник. Офсет на такой мелочи даёт неприемлемое число сходов.
  • На 1.5–2″ — одинарник предпочтительнее, офсет допустим в коряжнике/траве.
  • На 2.5–3″ — паритет. Одинарник для открытой воды, офсет для коряжника и крупной рыбы.
  • На 4–5″ — преимущественно офсет; одинарник используется на джиг-головке без шарнира.
  • Для червей и слагов — либо ультра-тонкий одинарник, либо удлинённый офсет (Decoy Worm 9).
  • Для раков и креатур — Wide Gap офсет обязателен из-за объёмного тела.

Источники данных

  • Производители: Decoy (Katsuichi Co., Ltd.), Owner America — каталожные данные по размерам и спецификациям.
  • Размерные ряды: BestAngler.com, SportFishTackle.com, Tackle Warehouse.
  • Связки «приманка ↔ офсет»: Дмитрий Савин, «Путеводитель по Decoy» (midacompany.ru).
  • Методология выбора крючков для микроджига: Николай Дунаев, SpinningSpb.ru.
  • Дополнительный материал: fmagazin.ru, fishingsib.ru, fspinning.ru.

Ограничения

  • Связки «конкретный размер офсета ↔ конкретная модель приманки» верифицированы в основном для Keitech. Для других брендов — экстраполяция по размеру и форме тела.
  • Грузы для дроп-шота и отводного приведены как ориентир — в этих монтажах груз и крючок разнесены, и логика подбора отличается.
  • Список эталонных приманок не претендует на исчерпывающий — это узнаваемые модели для ориентира.
  • Все рекомендации — стартовая точка, не догма. Реальная рыбалка корректирует подход.

## 📋 Очередь

### 2. Справочник хищных рыб
Карточки с полным профилем каждой хищной рыбы:
- Внешний вид
- Повадки и стиль охоты
- Сезонное поведение
- Рацион и пищевые предпочтения
- Признаки присутствия на водоёме
- Что нужно знать для эффективной ловли

Потенциально — отдельный аналогичный справочник по мирным рыбам.

### 3. Советник по сессии
Персональный план на рыбалку на основе условий:
- Тип водоёма
- Освещённость
- Атмосферное давление
- Температура воды + динамика за неделю
- Сезон

На выходе: рекомендации по приманке, горизонту ловли, проводке, тактике.

### 4. Подбор шнура
Лёгкий инструмент. На входе — условия ловли, на выходе — рекомендации по типу, толщине и характеристикам шнура.
